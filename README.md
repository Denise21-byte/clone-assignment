# Parallax Website Assignment

 Overview

Single-page parallax website based on the **W3Schools Parallax Template**, built to demonstrate **advanced CSS**, **semantic HTML**, and **responsive design**.

## Learning Outcomes (Rubric Aligned)

*  Semantic HTML structure
*  Parallax scrolling effect
*  Fixed/sticky navigation bar
*  Single Page Application (SPA)
*  Responsive layout (mobile-friendly)

##  HTML Requirements (Met)

*  One `index.html` file
*  Semantic sections: Header, About, Portfolio/Gallery, Contact, Footer
*  Anchor links for page navigation
*  At least **3 parallax sections**
*  Google Map iframe in Contact section
##  CSS Requirements (Met)

*  External stylesheet (`style.css`)
*  No inline styles
*  `background-attachment: fixed` used
*  Google Fonts applied
*  Media queries for responsiveness
*  Multiple CSS classes used
*  Pseudo-classes (e.g. `:hover`)

 File Structure
index.html
style.css
images/

 How to View

Open `index.html` in a web browser and scroll the page to view the parallax effect.
